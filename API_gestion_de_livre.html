Tutoriels interactifs Odoo (appelés "Tours")
Ces tutoriels guidés permettent de découvrir étape par étape les fonctionnalités d’un module Odoo (ex : CRM, ventes, inventaire) directement dans l’interface de l’application.
Intérêt principal :
- Formation rapide des utilisateurs sans besoin de documentation externe.
- Démonstration claire des fonctionnalités pour un jury ou des clients.
- Expérience interactive : l’utilisateur clique et apprend en même temps.
